 ساخت یک شمارنده (Counter) که فقط با this کار کند
یک آبجکت بسازید که شمارنده‌ای داشته باشد. این شمارنده باید مقدار اولیه 0 داشته باشد و هر بار که متد increment اجرا می‌شود، مقدار آن را افزایش دهد.

 محدودیت: فقط می‌توان از this برای دسترسی به مقدار شمارنده استفاده کرد. نباید از متغیرهای سراسری (let count;) یا متغیرهای داخل تابع (var counter;) استفاده کنیم
 یک آبجکت به نام taskManager بسازید که بتواند لیستی از کارها (tasks) را مدیریت کند. این آبجکت باید ویژگی‌ها و متدهای زیر را داشته باشد:

1.⁠ ⁠tasks: آرایه‌ای که کارها را ذخیره می‌کند.


2.⁠ ⁠addTask(task): یک متد که یک کار جدید را به لیست اضافه می‌کند.


3.⁠ ⁠showTasks(): یک متد که تمام کارهای موجود را در کنسول نمایش می‌دهد.


4.⁠ ⁠removeTask(index): یک متد که یک کار را با استفاده از ایندکس حذف می‌کند.
فقط باید از this استفاده کنید برای این تمرین
