لیست تسک‌های هفته

هفته اول

  • ساخت یک شمارنده (Counter) که فقط با this کار کند
  • یک آبجکت بسازید که شمارنده‌ای داشته باشد. مقدار اولیه ۰ و متد increment مقدار را افزایش دهد.
  • نباید از متغیرهای سراسری یا متغیرهای داخل تابع استفاده کنید.
  • ساخت آبجکت `taskManager` برای مدیریت لیستی از کارها
  • متد `addTask(task)` برای افزودن تسک جدید
  • متد `showTasks()` برای نمایش تمام تسک‌ها در کنسول
  • متد `removeTask(index)` برای حذف یک تسک با ایندکس مشخص
